Kapitel 0: Einführung

Schichtenmodell

  • Unterteilung des komplexen Kommunikationsvorgangs

    • Niedrigere Schichten bieten höheren Schichten Dienste an
    • Höhere Schichten nehmen Dienste der jeweils niedrigeren Schicht in Anspruch

  • Abstraktion von der Implementierung einer Schicht

    • Festlegung, welche Dienste angeboten werden, aber nicht wie sie erfüllt werden

Das ISO/OSI-Modell

  • ISO = International Organization for Standardization
  • OSI = Open Systens Interconnection
  • Unterteilt die Architektur des Systems, das die Kommunikationsfunktionalität erbringt, in insgesamt 7 Schichten.

  • IDU: Interface Data Unit
  • SDU: Service Data Unit
  • ICI: Interface Control Information
  • PCI: Protocol Control Information, in Form eines Headers den Nutzdaten vorangestellt
  • PDU: Protocol Data Unit, welcher auf der N-Schicht
    • die Nutzdaten der (N+1)-Schicht, SDU sowie
    • Protokolsteuerungsinformationen PCI der N-Schicht
      bezeichnet.

PDUs einiger Schichten haben eigene Bezeichnungen:

  • Segmenten auf der Transportschicht
  • Paketen auf der Vermittlungsschicht
  • Rahmen(Frames) auf der Sicherungsschicht

Einige Protokollmechanismen sind nicht klar einer bestimmten Schicht zuzuordnen, bzw. arbeiten sogar auf mehreren Schichten(Cross Layer)

Kapitel 1: Physikalische Schicht

Signale, Information und deren Bedeutung

Signale,Symbole

Signale sind zeitabhängige und messbare physikalische Größen. Definierten messbaren Signaländerungen lässt sich ein Symbol zuordnen. Diese Symbole repräsentierne Information.

bsp. Licht, Morsezeichen, Telegraphie, Sprache, Musik

Informationsgehalt

Der Informationsgehalt eines Zeichens oder Symbols drückt aus, wieviel Information durch das Zeichen übertragen wird.

Eigenschaften des Informationsgehalt:

  • Je seltener ein Zeichen auftritt, desto höher ist sein Informationsgehalt
  • Der Informationsgehalt einer Zeichenkette ist die Summe der Informationsgehalte der einzelnen Zeichen
  • Der Informationsgehalt eines vorhersagbaren Zeichens ist 0

Information

Information besteht in der Unsicherheit, Veränderungen eines Signals vorhersagen zu können.

I(x)=−log⁡2p(x) mit [I]=bitI(x) = -\log_{2}p(x) \text{ mit } [I] = \text{bit} I(x)=−log2​p(x) mit [I]=bit

Entropie

Den mittleren Informationsgehalt einer Quelle bezeichnet man als Entropie

H(x)=∑x∈χp(x)I(x)=−∑x∈χp(x)log⁡2(p(x)).H(x) = \displaystyle\sum_{x \in \chi}p(x)I(x) = -\displaystyle\sum_{x \in \chi}p(x)\log_{2}(p(x)). H(x)=x∈χ∑​p(x)I(x)=−x∈χ∑​p(x)log2​(p(x)).

Beispiel:

  1. Deterministische, diskrete Quelle, welche stets das Zeichen ‘A’ emittiert:

  2. Binäre Quelle, welche auf nicht vorhersehbare Weise die Zeichen ‘0’ oder ‘1’ emittiert:
    Qundefinedx011010100...Q \xrightarrow{x} 011010100...Qx​011010100...
    I(0)=−log⁡2(Pr[X=0])=−log⁡2(0.5)=1I(0) = -\log_{2}(Pr[X = 0]) = -\log_{2}(0.5) = 1I(0)=−log2​(Pr[X=0])=−log2​(0.5)=1 bit
    I(1)=−log⁡2(Pr[X=1])=−log⁡2(0.5)=1I(1) = -\log_{2}(Pr[X = 1]) = -\log_{2}(0.5) = 1I(1)=−log2​(Pr[X=1])=−log2​(0.5)=1 bit

Bedingte Entropie

Die bedingte Entropie

H(Y∣X)=∑x∈χp(x)H(Y∣X=x)=−∑x∈χp(x)∑y∈Yp(y∣x)log⁡2(p(y∣x))H(Y|X) = \displaystyle\sum_{x \in \chi}p(x)H(Y|X = x) = -\displaystyle\sum_{x \in \chi}p(x)\displaystyle\sum_{y \in Y}p(y|x)\log_{2}(p(y|x)) H(Y∣X)=x∈χ∑​p(x)H(Y∣X=x)=−x∈χ∑​p(x)y∈Y∑​p(y∣x)log2​(p(y∣x))

entspricht der verbleibenden Unsicherheit über Y, wenn X bekannt ist.

  • Sind X, Y voneinander abhängig, dann idt die bedingte Entropie kleiner als im unabhängigen Fall.
  • Sind X, Y voneinander unabhängig und ist der Ausgang von X bereits bekannt, so bleit die Entropie von Y vollständig erhalten.

Verbundentropie

Sei die Verbunddichte zweier Zufallsvariablen X und Y:

p(x,y)=Pr[X=x∣Y=y]Pr[Y=y].p(x,y) = Pr[X = x|Y = y]Pr[Y = y]. p(x,y)=Pr[X=x∣Y=y]Pr[Y=y].

Dann ist die Verbundentropie

H(X,Y)=−∑x∈χ∑y∈Yp(x,y)log⁡2p(x,y).H(X,Y) = -\displaystyle\sum_{x \in \chi}\displaystyle\sum_{y \in Y}p(x,y)\log_{2}p(x,y). H(X,Y)=−x∈χ∑​y∈Y∑​p(x,y)log2​p(x,y).

Transinformation

Die von Sender zu Empfänger über einen gedächtnislosen Kanal transportierte Information beyeichnet man als Transinformation(Mutual Information)

I(X;Y)=H(X)−H(X∣Y)=H(Y)−H(Y∣X).I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X). I(X;Y)=H(X)−H(X∣Y)=H(Y)−H(Y∣X).

Signal

Ein Signal transportiert Information. Erst durch eine Interpretationsvorschrift erhält diese Information eine Bedeutung, d.h. es muss eine Abbildung zwischen Symbolen(Signalwerten bzw. Wertebereichen) und Daten geben.
